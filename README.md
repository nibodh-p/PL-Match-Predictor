# Premier League Match Predictor

A data analysis project that uses Python to forecast the probability of Premier League match outcomes.

## About the Project
This project uses historical match data to:
* Calculate team performance metrics (Home/Away strength).
* Apply the **Poisson Distribution** to predict match outcomes.
* Identify patterns like "Road Warriors" and goal conversion rates.

## Files
- `investigation.ipynb`: The main notebook containing all the data analysis and prediction code.
- `premier_league_stats.csv`: The dataset used for the analysis.

## How to run
1. Clone this repository.
2. Ensure you have the required libraries installed:
   `pip install pandas numpy scipy`
3. Open `investigation.ipynb` in your preferred editor.
